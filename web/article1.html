在画面上显示字符

请在控制台画面上显示出相关字符

public class Hello {
        public static void main(String[] args){
            System.out.println("hello World!");
        }
    }

1.用来保存上述源程序的文件叫做源文件（source file）。 2.源文件的名称就是class后面书写的类名（Hello）加上扩展名.Java。
